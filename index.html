Known bug

We will list reported bug at issue page on Github (github/JDLabOrg/LemonEditor). For whom finding a bug, we will give about $30 (exactly, 30,000 Korean Won). For whom kill a bug, we will give about $100 (exactly, 100,000 Korean Won). This event will remain until all budget is used (2,000,000 Korean Won).

How to report bug.

  1. Find a reproducible bug.
  2. Check that is not on the list (issue page in GitHub)
  3. Make new empty project, and press record button (red dot button in top).
  4. Reproduce the bug.
  5. If that bug occurs, press record button again. A movie file may saved at your desktop.
  6. Send us email. Email should have - Basic computer spec (memory, cpu..), description of bug, and link of a saved movie file (please send us a cloud link to file, not attached file). Email link is iu@jdlab.org
  7. Note : We update bug list on 10 AM to 7 PM (Seoul Time Zone). Duplicated reporting does not counted. If there are duplicated reported bug on night time, only first reporter can get award.

How to kill a bug

Go to https://github.com/JDLabOrg/LemonEditor. Download it. kill it. Give us pull requests!.
