How to report bugs.

  1. Find a reproducible bug.
  2. Make sure it's a new one, with checking out the bug list.
  3. Make a new project, and start recording. A record button (a red dot) is located on top.
  4. Make the similar steps so as to reproduce the bug.
  5. When the bug appears, stop recording by press the record button again. A video clip will be at your desktop.
  6. Report the bug via iu@jdlab.org. The bug report requirement as follows:
    • Basic computer specs such as cpu, memory, and so on.
    • Bug descriptions
    • link of the video clip. Please upload your clip to somewhere else (such as dropbox, google drive, youtube, or any cloud server) and then share the link.
  7. Note: The bug list will be updated during working hours: 10Am~7Pm (KST) / 9Pm~6AM (PST). Duplicated bug will be ignored.
