How to report bugs.

  1. Find a reproducible bug. It should be checked on nightly release version
  2. Make sure it hasn't been discovered before, by checking the bug list.
  3. Create a new project, and start recording.
  4. Make the similar steps so as to reproduce the bug.
  5. When the bug appears, stop recording.
  6. Report the bug via iu@jdlab.org. The bug report requirement as follows:
    • Basic computer specs such as cpu, memory, and so on.
    • Bug descriptions
    • Attachment or Link of the video clip. - Attachment : Please attach video clip to e-mail. - Link : Please upload your clip to somewhere else (such as dropbox, google drive, youtube, or any cloud server) and then share the link.
  7. Note: The bug list will be updated during working hours: 10Am~7Pm (KST) / 9Pm~6AM (PST). Duplicated bug will be ignored.
